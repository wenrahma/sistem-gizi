# 📸 Tampilan Aplikasi

Berikut merupakan tampilan antarmuka **Sistem Pakar Gizi dan Pola Makan** yang dibangun menggunakan metode **Forward Chaining**.

---

## 🏠 1. Halaman Utama

Halaman awal aplikasi yang menampilkan informasi singkat mengenai sistem serta tombol untuk memulai proses konsultasi.

## ❓ 2. Halaman Konsultasi

Pengguna menjawab serangkaian pertanyaan mengenai kondisi kesehatan, pola makan, dan gaya hidup. Jawaban ini akan menjadi **fakta (working memory)** yang diproses menggunakan metode **Forward Chaining**.

  <img src="https://github.com/user-attachments/assets/5b91e38c-a45e-41bf-b352-6e7fcd5dcbe6" width="850">
</p>

---

## 🥗 3. Halaman Hasil Diagnosa

Setelah seluruh pertanyaan dijawab, sistem menampilkan:

- Kondisi gizi yang teridentifikasi
- Status kecocokan berdasarkan aturan
- Penjelasan kondisi
- Rekomendasi pola makan dan nutrisi

## 📄 4. Export Hasil

Hasil konsultasi dapat disimpan dalam format **.txt** sehingga pengguna dapat menyimpan atau mencetak hasil rekomendasi.

## ✨ Fitur Utama

- ✅ Sistem pakar berbasis **Forward Chaining**
- 🥗 Basis pengetahuan mengenai kondisi gizi
- ❓ Konsultasi interaktif berbasis pertanyaan Ya/Tidak
- 📋 Rekomendasi pola makan sesuai kondisi pengguna
- 📄 Export hasil konsultasi ke file **TXT**
- 🖥️ Antarmuka GUI yang mudah digunakan
